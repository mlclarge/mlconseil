Cet outil est proposé par [**Sprout Social**](https://sproutsocial.com/), une société qui édite une application pour community manager du genre Hootsuite ou Sociallymapp ! Cet utilitaire sert  , en mode gratuit , à  retoucher les images destinées à être diffusées sur les réseaux sociaux comme  **Twitter, Google+, facebook, pinterest, Instagram, Llinkedin, Youtube** ou en format personnalisable.

## ***Petit Hack pour faire connaître la suite Sprout Social***

Notons au passage cette petite technique de GrowthHacking, car sur la  requête « outil de retouche pour instagram » par exemple sur Google, Landscape, le nom de ce tool,  est très bien placé parmi les premiers résultats.

[![Landscape-Launch-Animation-Blog-Alt-Loop.gif](/images/blog/Landscape-Launch-Animation-Blog-Alt-Loop.gif) logiciel de retouche d'image en ligne

Pas de prise d’adresse mail, l’outil est en accès libre, sans aucune inscription.
## ***Conditions requises pour le redimensionnement***

 	- Les formats supportés sont jpg, gif non animé, png

 	- 10 Mo au plus pour les images

 	- Navigateur : disponible pour Google Chrome, Firefox, Safari and Internet Explorer version 11

## ***Tailles requises des images pour les réseaux sociaux***

Voici les tailles minimum pour **le redimensionnement qui s'effectue simplement en 4 étapes**.
### *Facebook*

 	- Image profil: 180 x 180 (Displays 170 x 170 on Desktop)

 	- Business Page Image profil: 180 x 180 (Displays 170 x 170 on Desktop)

 	- Couverture Photo: 820 x 312

 	- Partage Image: 1,200 x 630

 	- Partage Lien: 1,200 x 628

 	- Highlighted Image: 1,200 x 717 (Recommandé)

 	- Event Image: 1920 x 1080 (Recommandé)

[![0-482x300.jpg](/images/blog/0-482x300.jpg) Etape 1 reconnaissance de l'image et choix du réseau social
### ***Twitter***

 	- Profil Photo: 400 x 400 (Displays 200 x 200)

 	- En tête Photo: 1,500 x 500

 	- In-Stream Photo: Minimum 440 x 220 (2:1 Ratio)

### ***Instagram***

 	- Image profil: 110 x 110

 	- Photo Miniature: 161 x 161

 	- Photo Size: 1080 x 1080

 	- Instagram Stories: 1080 x 1920

[![1-502x285.jpg](/images/blog/1-502x285.jpg) Etape 2 choix du type de  redimension sur le ou les images précédentes selon son choix ici 2 images
### ***LinkedIn***

 	- Personnel Profile Image: 400 x 400 (Recommandé)

 	- Personnel Background Image: 1584 x 396

 	- Société Logo Image: 300 x 300

 	- Carré Logo: 60 x 60

 	- Bannière Image for société: 646 x 220 (Minimum)

 	- Hero Image: 1128 x 376

[![4-433x300.jpg](/images/blog/4-433x300.jpg) Etape 3 Retailler l'image ou cropping 
### ***Pinterest***

 	- Image profil: 165 x 165

 	- Pin  236 to 900 pixels

 	- Tableau  : 222 x 150 pixels (large miniature) ou 55 x 55 (plus petite miniature)

### **Google+**

 	- Profil Photo: 250 x 250 (Recommandé)

 	- Couverture Image: 1,080 x 608

 	- Partage Image – 497 x 373

 	- Partage Lien – 150 x 150 (Miniature)

 	- Partage Video: largeur de 496 Pixels

### YouTube

 	- chaîne Profil Image: 800 x 800

 	- chaîne Couverture Photo: 2,560 x 1,440

 	- Video téléchargements: 1280 x 720 (Minimum HD)

[![5-409x300.jpg](/images/blog/5-409x300.jpg) Etape 4 : Reste à downloader l'image retaillée en local pour publication 
### **Tumblr**

 	- Profil Photo: 128 x 128

 	- Image Pages: 500 x 750

Vous trouverez toutes les informations selon les plateformes ici : https://sproutsocial.com/insights/social-media-image-sizes-guide/.
