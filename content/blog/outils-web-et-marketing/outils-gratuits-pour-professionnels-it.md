Cyril Bois, responsable de la plateforme  service  Extendsclass.com  présente cette semaine son service d’ outils en ligne gratuits, plus d’une trentaine regroupés au seini d’une seule interface.
## ExtendsClass en bref, c'est quoi ?

[ExtendsClass](https://extendsclass.com) est une boîte à outils en ligne à destination des développeurs, des SEO analystes, et des webmasters. La plateforme propose exclusivement des outils en ligne, qui ne demandent ni installation ni inscription pour leurs utilisations, et sont entièrement gratuits.
## Pourquoi utiliser les outils de ExtendsClass ?

Les travailleurs de l'IT utilisent au quotidien des logiciels, souvent sous forme de clients lourds, à la fois ergonomiques et très pratiques pour une utilisation quotidienne. Mais pour une utilisation épisodique, les outils en ligne peuvent être une alternative intéressante, à la fois rapide d'utilisation (sans installation), et facile d'utilisation (pas de configuration).

## L'offre de ExtendsClass

### ***L’offre en quelques mots***

La plateforme ExtendsClass est récente (ouverture fin 2017), l'offre se complète au fur et à mesure et les outils sont constamment améliorés afin d'apporter de nouvelles fonctionnalités et d’apporter une meilleure expérience utilisateur.

Les outils proposés aident les professionnels de l'IT dans certaines tâches de leurs travails, nous allons voir le détail des différents outils mis à disposition.
### ***Outils pour les développeurs***

Les développeurs utilisent différents logiciels pour les aider dans leurs développements, ainsi que dans les phases de tests. ExtendsClass met à disposition des développeurs les applications en ligne suivantes :

- Clients REST / SOAP : Ce sont des clients HTTP qui permettent de tester les webservices de type REST ou SOAP.

Pour les webservices SOAP, le WSDL peut être chargé afin de générer les flux XML d'entrées des méthodes du webservice.

- Mock REST API : Cet outil permet de simuler une API REST, c’est utile quand le backend n'a pas encore été développé, cela permet d'avancer sur la partie front sans attendre la fin des développements de la partie backend.

- SQLite browser : C'est un interpréteur SQL qui permet d'importer et de requêter une base de données SQLite en quelques clics.

- Regex Tester : Les regex peuvent parfois donner envie de s'arracher les cheveux ... un petit outil permettant de faciliter les écritures d’expressions régulières.

- XPath Tester :  Analogue au Regex tester, cet outil facilite l'écriture d'expressions XPath.

Les développeurs ont également à leur disposition, des minifiers (HTML, CSS, JavaScript) et différents convertisseurs (JSON, CSV, base 64).
### ***Outils pour les webmasters***

La gestion de site web demande l'utilisation de certains outils pour optimiser le temps requis par certaines tâches. ExtendsClass propose les outils suivants aux webmasters :

- Générateur de sitemap : Ce générateur génère les sitemaps de type XML, qui peuvent être utilisés par les moteurs de recherche (Google, Bing, ...).

- Générateur de fichier robots.txt : Les fichiers robots.txt donnent des instructions aux moteurs de recherche, ce qui peut être utile pour le référencement.
### *Outils pour les SEO analystes*

Les SEO analystes ont à disposition une foison d'outils sur Internet, mais une partie d'entre eux sont payants. ExtendsClass propose les outils gratuits suivants :

- Analyse on-page de site web : Cet analyseur liste les aspects à améliorer au niveau SEO d'un site Internet afin de perfectionner son référencement.

- Vérificateur de liens hypertextes : L'échange de lien peut améliorer dans certains cas le référencement des sites Internet. Cet outil permet de vérifier que les liens retours soient bien toujours présents.
