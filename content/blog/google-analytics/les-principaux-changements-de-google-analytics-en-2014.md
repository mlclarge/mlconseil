Pour bien commencer l’année 2015,  prenons un instant pour revoir les **principaux changements intervenus  en 2014 sur la plateforme Google analytics,.**  Alors que l’année 2013 aura vu selon Search Engine Watch plus de 70 changements*, l’année 2014 a été marquée principalement par la sortie du statut de  version bêta  de Google analytics **universel**, par l’amélioration constante des possibilités du **remarketing,** par l’enrichissement du **reporting sur les conversions**  et enfin par le développement toujours plus  intuitif du gestionnaire de balises (**tag manager**).  Je préviens, je me suis encore lâché sur les screenshots (principalement pour les rapports :) ) !

## **Evolution de l'Interface de GA**

### **Codes  de suivi **

*GA Universel  perd son statut de bêta (lancé fin 2012)*

	- **Analytics Universal  **en standard : ajouts des données socio-démographiques, catégorielles  pour les sites web sous UA mais également pour les applications mobiles : [http://analytics.blogspot.fr/2014/09/enhanced-google-analytics-audience.html](http://analytics.blogspot.fr/2014/09/enhanced-google-analytics-audience.html)

### *Tag Ecommerce et remarketing*

	- Ecommerce amélioré (enhanced)  pour le tracking des **conversions panier** : suivi du comportement de l’acheteur sur le tunnel de vente (abandons, retours etc..)

[*](https://www.mauricelargeron.com/wp-content/uploads/2015/01/enhanced-ecommerce.jpg) Suivi comportemental Tunnel de vente

	- Intégration du **reciblage dynamique** : [http://analytics.blogspot.fr/2014/03/tailored-ads-better-results-dynamic.html](http://analytics.blogspot.fr/2014/03/tailored-ads-better-results-dynamic.html)

	- Constitution de **listes de remarketing** **« intelligentes » **: [http://analytics.blogspot.fr/2014/04/smarter-remarketing-with-google.html](http://analytics.blogspot.fr/2014/04/smarter-remarketing-with-google.html)

	- Elargissement du **remarketing dynamique** et paramétrages : [https://support.google.com/analytics/answer/3455585](https://support.google.com/analytics/answer/3455585)

### Nouvelles fonctionnalités*

	- Affichage de **Compteurs sur  disponibilités** dans l'admin. : 100 comptes GA,  50 propriétés par comptes  et 25 vues par propriété

[*](https://www.mauricelargeron.com/wp-content/uploads/2015/01/decompte.jpg) Enfin un décompte sur les entités disponible dans GA

	- **Indications « diagnostics** » : sous forme de notifications qui soulignent les imperfections de configuration  d’un compte.

[![diagnostics-google-analytics-288x300.jpg](/images/blog/diagnostics-google-analytics-288x300.jpg) Diagnostics Compte Google analytics

	- Association **GA et Adwords intégré**e : facilité dans l’association de plusieurs comptes Adw. sur une propriété

[![adwords-et-ga-310x158.jpg](/images/blog/adwords-et-ga-310x158.jpg) Association Adw et GA

	- Amélioration sur l’importation des **données personnalisées** : élargissements sur la possibilité d’importer des dimensions produits et autres statistiques métier en plus de celles standards de GA : [https://support.google.com/analytics/answer/3191417?hl=fr](https://support.google.com/analytics/answer/3191417?hl=fr)

	- **Groupement de contenus** : faire des ensembles de pages uniques, des groupes thématiques pour simplifier l’analyse : [http://analytics.blogspot.fr/2014/05/introducing-content-grouping-in.html](http://analytics.blogspot.fr/2014/05/introducing-content-grouping-in.html)

	- **Segmentation Acquisition "Marque & Générique"** : Définir dès le départ les termes d’acquisition qui segmenteront les canaux d’acquisition, c’est l’objectif de cette fonctionnalité : [http://analytics.blogspot.fr/2014/06/segmenting-brand-and-generic-paid.html](http://analytics.blogspot.fr/2014/06/segmenting-brand-and-generic-paid.html)

[![segmenation-aquisition-marque-et-generique-310x237.jpg](/images/blog/segmenation-aquisition-marque-et-generique-310x237.jpg) Segmentation Acquisition "Marque & Générique"
## **Arrivée de nouveaux Rapports **

### Audience*

	- Implémentation de l’User Id  dans une vue dédiée : cette nouvelle dimension, transversale à GA, sans identification personnelle, qui permet de suivre les conversions provenant de divers appareils

[*](https://www.mauricelargeron.com/wp-content/uploads/2015/01/userId-multidevices-dans-ga.jpg) Analyse chemin Multi-devices

	- **Analyse comparative** : selon la catégorisation du site, Google divulgue les performances tierces d’autres site web avec pas moins de 1700 métiers.

[![benchmark-310x171.jpg](/images/blog/benchmark-310x171.jpg) Analyse comparative métier (Benchmarking)

	- **Utilisateurs uniques** : une segmentation toute faîte par période..

[![utilisateur-uniques-310x226.jpg](/images/blog/utilisateur-uniques-310x226.jpg) Rapports Utilisateurs uniques

	- **Version  Premium **: des rapports agrégés de différentes propriétés

[![aggregation-rapports-premium-310x297.jpg](/images/blog/aggregation-rapports-premium-310x297.jpg) Fonction "Roll-up" Aggrégation de Propriétés GA
### Acquisition*

	- **Display** : tous les critères de ciblages apparaissent désormais (mots clés, emplacements, intérêts, âges etc…)

[*](https://www.mauricelargeron.com/wp-content/uploads/2015/01/ciblage-du-réseau-display.jpg) Ciblage du réseau display

	- **Vidéo** : une vue roiste des campagnes sur Youtube

[![campagne-video-310x169.jpg](/images/blog/campagne-video-310x169.jpg) Rapport Campagne Vidéos

	- **Campagnes Shopping** : une segmentation par catégorie, produit et par marque permet d’identifier la  performance l’audience ciblée.

[![shopping-campagne-310x183.jpg](/images/blog/shopping-campagne-310x183.jpg) Rap. Campagnes Shopping
### Comportement*

	- **Flux de comportement** : amélioré avec la fonctionnalité de regroupement de contenus.

[![flux-comportement-310x290.jpg](/images/blog/flux-comportement-310x290.jpg) Flux de comportements sur Groupes de contenus
## **Gestionnaire de balises**

Une plateforme qui mériterait une rétrospective dédiée ! 2014 a vu de nombreuses améliorations.  Je rappelle qu’elle permet de centraliser,  via l’ajout d’un conteneur  à la place du script de base GA,  tous les marqueurs (balises) d’un site web.

[![tagmanager-310x247.jpg](/images/blog/tagmanager-310x247.jpg) Nouvelle interface Tag Manager

	- **50 ressources disponibles** pour mieux appréhender le tag manager : [https://www.mauricelargeron.com/tutoriaux-liens-pour-comprendre-le-gestionnaire-de-balises/](https://www.mauricelargeron.com/tutoriaux-liens-pour-comprendre-le-gestionnaire-de-balises/)

	- **Display** disponible pour UA

	- **Test A/B**  pour les applications mobiles ok

	- Tracking avancé ecommerce (enhanced)

	- **Priorisation du déclenchement** de l’ordre des balises

	- **Template  ‘marchand de confiance »** dispo. pour la France

	- **Api** disponible  pour les programmeurs : [https://developers.google.com/tag-manager/api/v1/reference/](https://developers.google.com/tag-manager/api/v1/reference/)

	- Une **nouvelle interface** plus "design" et  intuitive pour le marketeur (img)

	- **Import et export des containers** (entre autre permet :  le traitement en lots, la migration de l’ancienne interface vers la nouvelle)
