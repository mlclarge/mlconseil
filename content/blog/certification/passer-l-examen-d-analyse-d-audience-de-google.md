Dans le cadre du programme Google Partner, je viens de repasser l'examen d'**agrément à titre personnel ,** cela faisait un moment... Comme j’ai pu le faire pour la certification adwords, je donne ici les premiers éléments pour s'y préparer, les thèmes abordés,  et un extrait de quelques questions. ## **Cursus de préparation du QCM**

**Deux  cours** vous attendent pour [préparer la certification](https://www.mauricelargeron.com/debrief-sur-la-certification-google-analytics-4/) , une **version    Française**  sous forme d'aide en ligne et **une version anglaise** mais sous forme d'**académie** avec comme professeur l'illustre **Justin Cutroni**. A l’époque , en 2011,donc il n’y a pas si longtemps , le QCM était en anglais uniquement…Surprise, c’est la langue de molière qui l’emporte cette fois, c'est pas plus mal. D'une manière générale, l'exercice est vraiment abordable, pas technique du tout, maitriser simplement le taguage des liens de campagne avec les variables UTM  est un plus (5 questions).

## **Contenus des questions de l'examen (thèmes et extraits)**

C’est un QCM de 70 questions, 90 minutes, et 80% de réussite est le minimum requis. Voici résumé les thématiques principales des questions du moment :

[![Image](/images/blog/repartition-310x153.jpg)](https://www.mauricelargeron.com/wp-content/uploads/2014/11/repartition.jpg) **Thématiques de la certification**

Exemple 1  : **Quelles actions les filtres vous permettent-ils d’effectuer ?**

• Toutes ces réponses (oui)
• Inclure des données dans une vue
• Exclure des données d’une vue
• Modifier la façon dont les données s’affichent dans vos rapports

Exemple 2 :  **Parmi les propositions suivantes, laquelle constitue un avantage offert par la mise en œuvre de google gestionnaire de balises ?**
1. Vous pouvez ajouter des balises google analytics à votre site sans modifier le code de ce dernier
2. Toutes les réponses sont correctes (oui)
3. Vous pouvez modifier les valeurs de configuration dans votre application mobile sans recréer de binaire.
4. Vous pouvez ajouter des balises tierces à votre site sans modifier le code de ce dernier
5. Vous pouvez ajouter des balises adwords à votre site sans modifier le code de ce dernier

Exemple 3 : **La configuration d’objectifs vous permet d’obtenir**
1. La liste des transactions
2. Le CA issu du commerce électronique
3. Les taux de conversion (oui)
4. Le taux de rebond

Exemple 4 :**Parmi les éléments suivants, lesquels constituent des types d’appel faisant l’objet d’un suivi par google analytics ?**

1. Tous ces éléments (oui)
2. Les évènements
3. Les pages vues
4. Les transactions
Exemple 5 : **Vos rapports sur les entonnoirs multicanaux ne comportent pas de données. Quelle est l’explication la plus probable ?**
1. Vous n’avez pas activé les données démographiques
2. Vous n’utilisez pas Google gestionnaire de balises
3. Vous n’avez pas configuré d’entonnoirs de conversion vers l’objectif
4. Vous n’utilisez pas les tests de pages de contenus
5. Vous n’avez pas mis en œuvre les objectifs ou le commerce éléctronique (oui)

Bon, je suis pas sûr d’avoir bien répondu à toutes les questions car je n’ai pas eu 100% de réussite (eh eh…je vous dirais  pas combien mais j'y suis passé du 1er coup !). Bonne révision et bonne chance !
